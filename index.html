About the game

This is a minimalist platformer in which the only control is to Jump, made for the Ludum Dare 48-hour Challenge #26. You must ascend to the top by jumping and utilizing springs, switches and platforms.

To run on Linux, you must have libsfml-system, libsfml-window, and libsfml-graphics installed. To install these on Ubuntu, run the following as root (sudo):

 # apt-get install libsfml-*

How to build from source

Linux

Make sure you have the development libraries for SFML installed, as well as CMake, and at least version 4.7 of g++.

$ git clone https://github.com/Furyhunter/ld26-minimalism.git
$ cd ld26-minimalism
$ mkdir build
$ cd build
$ cmake ../
$ make

Windows

When configuring the project in CMake, manually set the libsfml paths in the cache. You need to use mingw32 4.7 or Visual Studio 2012 to build the project as the code makes extensive use of c++11 features.
